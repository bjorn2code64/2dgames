# ShapeShifter2D

## C++ 2D Game Engine for Windows + Games

### Libraries

- ss2dtest - The main program.
- wrap32lib-d2d - Direct2D
- wrap32lib-extras - Extras that depend on the other wrap32 libraries
- wrap32lib-file - File
- wrap32lib-gui - Window / UI
- wrap32lib - Base and Utility
